总体均值区间估计

有一大批糖果，从中随机抽取16袋，称得质量（单位：g）如下：

506 508 499 503 504 510 497 512

514 505 493 496 506 502 509 496

设袋装糖果质量近似服从正态分布，试求总体均值$\;\mu$的置信度为0.95的置信区间。

假设与条件:

1.糖果质量近似服从正态分布。

2.随机取16袋样本，置信度为0.95，构造估计量的样本数小于30，根据经验为小样本,并且总体方差未知。

要用t分布建立总体均值$\;\mu$的置信区间

解：总体均值$\mu$在1-$\alpha$置信水平下的置信区间为$\big(\overline{X} \pm \frac{S}{\sqrt{n}}t_{\alpha/2}(n-1)\big) $

$\overline{X}$—样本均值，S为样本标准差，n为样本容量。

显著水平$\alpha$=0.05,$\alpha/2$=0.025,自由度n-1=15，查t分布表得$t_{\alpha/2}(n-1)$=2.1315

报童售报模型

基本问题是报童每天用每份2元价格进一批报纸，以4元零售，当天没卖掉的报纸退回，得到每份1元的补偿，报童每天应该进多少报纸使日均利润最高。

由于真实情况供应链模型比较复杂，要考虑很多因素，比如供货商与零售商决策之间的影响，价格对市场的影响等。还有这个问题中报纸是个易逝品，假设当天报纸没卖完就会损耗，因为没人看前一天的报纸。

所以这里的模型都是在最理想的情况下得到的。

连续型需求下的模型：
假设报童每售出一份报纸获利2元,记为s1=2.剩余退回损失为1元，记为s2=1

前面假设需求量为离散型随机变量，当报纸需求量很大时，用离散随机变量表示概率分布和求期望会很麻烦。

现在根据以往统计资料及经验，把需求量看做连续型随机变量，假设报纸需求量大致服从正态分布，设其平均值为260，标准差为50，即需求量服从N(260,50^2)的正态分布。

建模：

由于是连续型随机变量，就用概率密度p(r)来表示需求量。

同时连续型随机变量不考虑某一点的概率，而是考虑随机变量落在区间的概率，所以用p(r)dr替代离散模型中的f(r)。

可以得日均利润公式为

问题转变成了求函数最大值的问题，这里就是要求最优采购量q,使得日均利润E(q)最大化

对q求导数并令$\frac{dE}{dq} =0$:

$E(q) = \int_{-\infty}^q [s_1r - s_2(q-r)]p(r)dr + \int_{q}^{\infty}s_1qp(r)dr$

= $\int_{-\infty}^qs_1rp(r)dr$ - $\int_{-\infty}^qs_2qp(r)dr$ +$\int_{-\infty}^qs_2rp(r)dr$ +$\int_{q}^{\infty}s_1qp(r)dr$

= $s_1r\int_{-\infty}^qp(r)dr$ - $s_2q\int_{-\infty}^qp(r)dr$ +$s_2r\int_{-\infty}^qp(r)dr$ +$s_1q\int_{q}^{\infty}p(r)dr$

$\frac{dE(q)}{dq} = s_1qp(q) - s_2\big(\int_{-\infty}^qp(r)dr + qp(q)\big)+s_2qp(q) + s_1(\int_{q}^{\infty}p(r)dr-qp(q)) \big)$

=$s_1qp(q) - s_2\int_{-\infty}^qp(r)dr - s_2qp(q)+s_2qp(q) + s_1\int_{q}^{\infty}p(r)dr-s_1qp(q) $

=$ s_1\int_{q}^{\infty}p(r)dr - s_2\int_{-\infty}^qp(r)dr$

因为概率密度性质：$\int_{-\infty}^{\infty} p(r)dr = 1$ ,得$s_1\int_{q}^{\infty}p(r)dr = s_1(1 - \int_{-\infty}^{q}p(r)dr) $

最终$\frac{dE(q)}{dq} = s_1 - (s_1+s_2)\int_{-\infty}^{q}p(r)dr$ = 0

然后要证明有最大值，继续求二阶导数：

$\frac{d^2E(q)}{dq^2}$ <0,并且q为驻点，所以在$\frac{dE(q)}{dq} = 0$时E(q)有最大值

可以记r的分布函数为$F(q)=\int_{-\infty}^{q}p(r)dr$，F(q)也是累积分布函数

得到：$\frac{dE(q)}{dq} = s_1 - (s_1+s_2)\int_{-\infty}^{q}p(r)dr$

=$s_1 - (s_1+s_2)F(q)$ = 0

由前面的条件s1=2,s2=1时,
$F(q) = \frac{s_1}{s_1+s_2} = \frac{2}{3}$

因为$\frac{dE(q)}{dq} = s_1 - (s_1+s_2)F(q) =0$时，E(q)有最大解,
所以相当于$F(q) = \frac{s_1}{s_1+s_2} $时，E(q)有最大解.

由累积分布函数与密度函数的关系可得：

$F(q) = P\{R\in(-\infty,q)\}=\int_{-\infty}^{q}p(r)dr = \frac{s_1}{s_1+s_2} =\frac{2}{3}$,这个式子的意思是：有一个q值，使得r 属于区间$(-\infty,q)$的概率等于$\frac{2}{3}$。

换句话说,就是这个q值，使得分布函数F(q)=$\frac{2}{3}$时，E(q)有最大解,现在就是要求出这个q值，本案例中就是每天进货量。

由于前面假设r是服从正态分布的，于是$F(q)= \int_{-\infty}^{q}p(r)dr$

$= \frac{1}{\sigma\sqrt{2π}}\int_{-\infty}^{q}e^{-(r^{,}-\mu^{2})/(2\sigma^{2}))}dr^{,}$

$=\frac{1}{2}\big[ 1+ erf\big( \frac{q-\mu}{\sigma\sqrt{2}} \big) \big] = \frac{2}{3}$

汽车选购问题

多属性决策：最优方案的选择，或者按照方案优劣排序，或者给出优劣程度的数量

多属性决策方案优劣由若干属性（如汽车价格，性能，款式等因素）给以定量或定性表述

属性集合选择原则：

1.全面考虑影响决策目标的因素，选择影响力较强的属性;

2.各属性相关性不要太强

3.尽量选择定量(例如价格等，数值属性)属性，定性(性能，款式等分类)属性要能明确分出优劣程度

4.某个属性对各备选方案差别很小，根据该属性难以辨别方案优劣，这个属性不必入选

5.属性太多时，应该分层，上层每一属性包含下层若干子属性

决策矩阵

假设3种型号汽车，记做A1,A2,A3。汽车3种属性分别为价格，性能和款式记做X1,X2,X3。

其中性能和款式用打分的方式判断优劣，满分为10分。

矩阵元素$d_{ij}$表示$A_{i}$对$X_{j}$的取值，或称为原始权重

由于决策矩阵属性值物理意义会有不同，所以在分析前要进行标准化处理，标准化处理要区分效益型属性，费用型属性和区间型属性，例如汽车性能，款式为效益型属性，其属性值越大，对决策的重要程度越高。汽车价格则是费用型属性，费用越低，对决策的重要程度越高。

上面决策矩阵明显效益属性占多数，所以先把费用属性（汽车价格）标准化，统一成效益型属性。

汽车价格是逆指标，它的值越小越好，它的作用方向与效益型属性不同，这里就要改变逆指标的性质和作用方向，以达到与效益型属性相同作用方向。这里使用取倒数法

然后统一进行标准化，前提假设是属性值对决策的重要性是线性关系。因为这里是比较三个方案优劣，所以使用列向量归一化处理，

$r_{ij} = \frac{d_{ij}}{\sum_{i=1}^m d_{ij}}$

有了归一化后的属性矩阵，下面就要确定各属性的权重，也就是各个属性对决策目标的影响程度。属性权重可以人为根据检验主观判断，也可以偏于客观判断

客观的判断方法有信息熵法：

如果把归一化矩阵各列向量看做信息量的概率分布，那么各型号汽车中某个属性值趋于一致，就是概率差别不大，提供的信息熵就越大，就无法进行优劣的判断。

相反如果信息熵越小，说明提供的信息量越少，不确定性越小，就容易进行优劣的判断

结合本案例说明：

比如观察上面归一化后的矩阵第三列，就是汽车款式属性这一列，其差别不大，基本趋于一致，说明这个属性信息熵大，不能通过这个属性比较汽车的忧劣。

这只是通过观察，实际要通过对各属性熵的计算结果进行比较。

各列属性信息熵计算公式：

$E_j$越大，说明属性相差不大，则这个属性贡献越低,对汽车优劣辨别作用不大，反之是值越小贡献越高。所以这里定义属性X的区分度：
$F_j=1-E_j,\;0\leq F_i \leq 1$.

然后使用归一化的区分度计算各属性权重：
$w_i = \frac{F_j}{\sum_{j=1}^n F_j}\;,j=1,2,..n$

第三种是接近理想解的偏好排序法（topsis法）

理想解法基本思路是，从属性值中构造正理想解与负理想解，然后计算备选方案与理想解的相对接近度，从而判断各备选方案的优劣。

相对接近是指同时考察备选方案与正理想解，负理想解之间的距离,最优方案是离正理想解距离近同时与负理想解距离远的方案。

关于假设检验结果解释
首先前三个是有方向的检验，通过一个比较苛刻的显著性水平（0.05）来反驳H0，从而证明H1的成立,这里把想证明不正确的命题作为原假设

从结果看出吸烟孕妇比不吸烟孕妇的新生儿体重低,并且新生儿体重低的比例要高。而早产率的差异并不显著。早产与否对应着孕期的长短，所以
为了解决提出的第二个问题，即孕妇吸烟是否会使早产率增加，怀孕期长度是否对新生儿体重有影响。我们还需要进一步借助其他方法充分利用数据进行分析。

经过前面的参数估计和假设检验，可以看出吸烟对婴儿体重有着显著的影响，但是这有可能是多因素影响的结果，比如怀孕期，孕妇体重，早产率等。
所以这里要利用回归分析，建立婴儿体重与孕妇吸烟状况,怀孕期等因素的回归模型来具体分析影响新生儿体重的因素。

首先用一元线性回归分析吸烟孕妇新生儿体重与孕期的关系

假设用y和x分别表示新生儿体重和怀孕期两个变量，二者关系可以用一次函数$y = b_0+b_1x$来描述
首先计算系数$b_0和b_1$

$b_1=\frac{n\sum_{i=1}^nx_iy_i -\sum_{i=1}^nx_i\sum_{i=1}^ny_i}{n\sum_{i=1}^nx_i^2-(\sum_{i=1}^nx_i)^2}$

$b_0 = \overline{y} - b_1\overline{x}$,其中$\overline{y} = \frac{\sum_{i=1}^ny_i}{n},\overline{x}=\frac{\sum_{i=1}^nx_i}{n}$
