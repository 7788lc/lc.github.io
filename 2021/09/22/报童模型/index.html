报童售报模型

基本问题是报童每天用每份2元价格进一批报纸，以4元零售，当天没卖掉的报纸退回，得到每份1元的补偿，报童每天应该进多少报纸使日均利润最高。

由于真实情况供应链模型比较复杂，要考虑很多因素，比如供货商与零售商决策之间的影响，价格对市场的影响等。还有这个问题中报纸是个易逝品，假设当天报纸没卖完就会损耗，因为没人看前一天的报纸。

所以这里的模型都是在最理想的情况下得到的。

1.离散型需求下的报童模型：

我们将每天报纸需求看做离散随机变量，就是我们假设以需求量为单位，每100份报纸为1个单位，每

种需求量对应一个概率

需求量/单位 r 0 1 2 3 4 5
概率/分布 f(r) 0 0.1 0.3 0.4 0.1 0.1

建模：

定义：需求量取值为r时的概率为f(r)(r=0,1,2,..n),

报童每天进报纸数量为q单位,

报童每天售出1个单位报纸获利记为$s_{1}$,

剩余退回1单位报纸的损失为$s_{2}$,

每天利润与进货量(q)和需求量(r)是相关的，定义利润函数s(r,q)

每天利润分两种情况考虑:

*全部卖完,供不应求，利润为$q \times s_{1}$

*没卖完，供大于求,则损失为$(q-r)\times s_{2}$，获利为卖出量减损失量$ r \times s_{1} - (q-r)\times s_{2}$

连续型需求下的模型：
假设报童每售出一份报纸获利2元,记为s1=2.剩余退回损失为1元，记为s2=1

前面假设需求量为离散型随机变量，当报纸需求量很大时，用离散随机变量表示概率分布和求期望会很麻烦。

现在根据以往统计资料及经验，把需求量看做连续型随机变量，假设报纸需求量大致服从正态分布，设其平均值为260，标准差为50，即需求量服从N(260,50^2)的正态分布。

建模：

由于是连续型随机变量，就用概率密度p(r)来表示需求量。

同时连续型随机变量不考虑某一点的概率，而是考虑随机变量落在区间的概率，所以用p(r)dr替代离散模型中的f(r)。

可以得日均利润公式为

问题转变成了求函数最大值的问题，这里就是要求最优采购量q,使得日均利润E(q)最大化

对q求导数并令$\frac{dE}{dq} =0$:

$E(q) = \int_{-\infty}^q [s_1r - s_2(q-r)]p(r)dr + \int_{q}^{\infty}s_1qp(r)dr$

= $\int_{-\infty}^qs_1rp(r)dr$ - $\int_{-\infty}^qs_2qp(r)dr$ +$\int_{-\infty}^qs_2rp(r)dr$ +$\int_{q}^{\infty}s_1qp(r)dr$

= $s_1r\int_{-\infty}^qp(r)dr$ - $s_2q\int_{-\infty}^qp(r)dr$ +$s_2r\int_{-\infty}^qp(r)dr$ +$s_1q\int_{q}^{\infty}p(r)dr$

$\frac{dE(q)}{dq} = s_1qp(q) - s_2\big(\int_{-\infty}^qp(r)dr + qp(q)\big)+s_2qp(q) + s_1(\int_{q}^{\infty}p(r)dr-qp(q)) \big)$

=$s_1qp(q) - s_2\int_{-\infty}^qp(r)dr - s_2qp(q)+s_2qp(q) + s_1\int_{q}^{\infty}p(r)dr-s_1qp(q) $

=$ s_1\int_{q}^{\infty}p(r)dr - s_2\int_{-\infty}^qp(r)dr$

因为概率密度性质：$\int_{-\infty}^{\infty} p(r)dr = 1$ ,得$s_1\int_{q}^{\infty}p(r)dr = s_1(1 - \int_{-\infty}^{q}p(r)dr) $

最终$\frac{dE(q)}{dq} = s_1 - (s_1+s_2)\int_{-\infty}^{q}p(r)dr$ = 0

然后要证明有最大值，继续求二阶导数：

$\frac{d^2E(q)}{dq^2}$ <0,并且q为驻点，所以在$\frac{dE(q)}{dq} = 0$时E(q)有最大值

可以记r的分布函数为$F(q)=\int_{-\infty}^{q}p(r)dr$，F(q)也是累积分布函数

得到：$\frac{dE(q)}{dq} = s_1 - (s_1+s_2)\int_{-\infty}^{q}p(r)dr$

=$s_1 - (s_1+s_2)F(q)$ = 0

由前面的条件s1=2,s2=1时,
$F(q) = \frac{s_1}{s_1+s_2} = \frac{2}{3}$

因为$\frac{dE(q)}{dq} = s_1 - (s_1+s_2)F(q) =0$时，E(q)有最大解,
所以相当于$F(q) = \frac{s_1}{s_1+s_2} $时，E(q)有最大解.

由累积分布函数与密度函数的关系可得：

$F(q) = P\{R\in(-\infty,q)\}=\int_{-\infty}^{q}p(r)dr = \frac{s_1}{s_1+s_2} =\frac{2}{3}$,这个式子的意思是：有一个q值，使得r 属于区间$(-\infty,q)$的概率等于$\frac{2}{3}$。

换句话说,就是这个q值，使得分布函数F(q)=$\frac{2}{3}$时，E(q)有最大解,现在就是要求出这个q值，本案例中就是每天进货量。

由于前面假设r是服从正态分布的，于是$F(q)= \int_{-\infty}^{q}p(r)dr$

$= \frac{1}{\sigma\sqrt{2π}}\int_{-\infty}^{q}e^{-(r^{,}-\mu^{2})/(2\sigma^{2}))}dr^{,}$

$=\frac{1}{2}\big[ 1+ erf\big( \frac{q-\mu}{\sigma\sqrt{2}} \big) \big] = \frac{2}{3}$

下面根据公式，使用程序求解：

281.4447252037101

结果可见当报童每天进281份报纸使可以获得最高利润

需要将q值带入$E(q) = \int_{-\infty}^q [s_1r - s_2(q-r)]p(r)dr + \int_{q}^{\infty}s_1qp(r)dr$公式,求出最高利润值

由假设知道r服从正态分布$N(260,50^2)$,

$p(r)= \frac{1}{\sigma\sqrt{2π}}e^{-(r^{,}-\mu^{2})/(2\sigma^{2})}dr$

对E(q)使用变量代换求积分即可

令$u=\frac{r-\mu}{\sigma \sqrt{2}}$

则$du = \frac{dr}{\sigma\sqrt{2}}, r=u\sigma\sqrt{2}+\mu$

积分上限 =$\frac{q-\mu}{\sigma \sqrt{2}}$,为方便计算暂时用q替代

$E(q)= \int_{-\infty}^q [s_1r - s_2(q-r)]p(r)dr + \int_{q}^{\infty}s_1qp(r)dr$

=$s_1\int_{-\infty}^qrp(r)dr$ - $s_2q\int_{-\infty}^qp(r)dr$ +$s_2\int_{-\infty}^qrp(r)dr$ +$s_1q(1-\int_{-\infty}^{q}p(r)dr)$

先分开计算

$s_1\int_{-\infty}^qrp(r)dr$ = $s_1\frac{1}{\sqrt{\pi}}\int_{-\infty}^q(\sqrt{2}\sigma u +\mu) e^{-u^2}du$

= $s_1(\frac{\sigma \sqrt{2}}{\sqrt{\pi}} \int_{-\infty}^q u e^{-u^2}du+\frac{\mu}{\sqrt{\pi}}\int_{-\infty}^q e^{-u^2}du )$

=$-s_1 \frac{\sqrt{2}\sigma e^{-q^2}}{2\sqrt{\pi}} -s_1\frac{\mu}{2\sqrt{\pi}}e^{-q^2}$

类似的

$s_2\int_{-\infty}^qrp(r)dr$ = $s_2\frac{1}{\sqrt{\pi}}\int_{-\infty}^q(\sqrt{2}\sigma u +\mu) e^{-u^2}du$

=$-s_2 \frac{\sqrt{2}\sigma e^{-q^2}}{2\sqrt{\pi}} -s_2\frac{\mu}{2\sqrt{\pi}}e^{-q^2}$

$s_2q\int_{-\infty}^qp(r)dr = -s_2q\frac{1}{2\sqrt{\pi}}e^{-q^2}$

$s_1q(1-\int_{-\infty}^{q}p(r)dr)$ =$s_1q+s_1q\frac{1}{2\sqrt{\pi}}e^{-q^2}$

则原式E(q) = $-s_1 \frac{\sqrt{2}\sigma e^{-q^2}}{2\sqrt{\pi}} -s_1\frac{\mu}{2\sqrt{\pi}}e^{-q^2}+s_2q\frac{1}{2\sqrt{\pi}}e^{-q^2} -s_2 \frac{\sqrt{2}\sigma e^{-q^2}}{2\sqrt{\pi}} +s_2\frac{\mu}{2\sqrt{\pi}}e^{-q^2}+s_1q+s_1q\frac{1}{2\sqrt{\pi}}e^{-q^2}$

由于$F(q)=\int_{-\infty}^qp(r)dr = -\frac{1}{2\sqrt{\pi}}e^{-q^2}$，

并且由P(q)=$\frac{1}{\sigma\sqrt{2\pi}}e^{-(q^{,}-\mu^{2})/(2\sigma^{2})}$，得$e^{-(q^{,}-\mu^{2})/(2\sigma^{2})} =P(q)\sigma\sqrt{2\pi}$

带入E(q)整理得：

E(q) =$s_1q-s_1qF(q)+s_1\mu F(q)-s_1\sigma^2p(q)-s_2qF(q)+s_2\mu F(q)-s_2\sigma^2p(q)$

465.4576921796353
